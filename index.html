Dictionary 类似于Map 利用Hash函数进行Key - Value 存储，因此具有很快的查找速度。 Set 和 Dictionary 相似 但是不储存Value,Key不重复

关键字参数：

可变参数在函数调用时，组装成tuple,关键字参数组装成dict. eg: def person(name, age, **kw): print('name:', name, 'age:', age, 'other:', kw) person('Adam', 45, gender='M', job='Engineer') name: Adam age: 45 other: {'gender': 'M', 'job': 'Engineer'} 参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数。 def f1(a, b, c=0, *args, **kw): 尾递归： return时候1.调用自己本身 2.不含表达式 切片： 经常取指定范围的操作，eg: L[0:3] 取list中的从0到2，不包括3 L[：：5]间隔为5的取 filter()函数返回的是一个Iterator，也就是一个惰性序列，所以要强迫filter()完成计算结果，需要用list()函数获得所有结果并返回list。 排序 sorted(L,key,reverse) 参数key代表对l中每一个值进行key代表的函数操作 ;reverse = TRUE代表反向排序 装饰器 假设我们要增强now()函数的功能，比如，在函数调用前后自动打印日志，但又不希望修改now()函数的定义，这种在代码运行期间动态增加功能的方式，称之为“装饰器”（Decorator） 实质上装饰器是一个返回函数的高阶函数 使用__slots__要注意，__slots__定义的属性仅对当前类实例起作用，对继承的子类是不起作用的
