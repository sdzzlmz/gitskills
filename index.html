Dictionary 类似于Map 利用Hash函数进行Key - Value 存储，因此具有很快的查找速度。 Set 和 Dictionary 相似 但是不储存Value,Key不重复

关键字参数：

可变参数在函数调用时，组装成tuple,关键字参数组装成dict. eg: def person(name, age, **kw): print('name:', name, 'age:', age, 'other:', kw) person('Adam', 45, gender='M', job='Engineer') name: Adam age: 45 other: {'gender': 'M', 'job': 'Engineer'} 参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数。 def f1(a, b, c=0, *args, **kw): 尾递归： return时候1.调用自己本身 2.不含表达式
